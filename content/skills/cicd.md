- Jenkins: configuración de pipelines de build para generar ejecutables e instaladores
- GitHub Actions: configuración de workflows de calidad de código y aspectos de seguridad (linters, análisis de seguridad), implementados tanto en proyectos personales (como Goagent) como en el ámbito profesional
- Despliegues (CD): en el trabajo actual se hacen de forma manual por política de seguridad de la empresa, sin automatización de entrega continua
