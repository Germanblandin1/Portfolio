Diseño e implemento agentes de IA (patrón ReAct), sistemas multiagente y de orquestación
(pipelines, grafos dinámicos, patrón supervisor), Retrieval-Augmented Generation (RAG) y el
protocolo Model Context Protocol (MCP). En el día a día uso herramientas como Claude Code, GitHub
Copilot, Codex, Cursor y GitHub Spec Kit, con una metodología de Spec Driven Development (SDD)
que prioriza la planificación antes de ejecutar, gestión de contexto para minimizar alucinaciones,
y combinación de skills, MCPs y sub-agentes según el caso. También trabajo con datos: pipelines
completos de análisis, desde la exploración inicial hasta la comparación de modelos, limpieza y
preparación de datos, usando Pandas, NumPy, Scikit-learn y Jupyter Notebook.
